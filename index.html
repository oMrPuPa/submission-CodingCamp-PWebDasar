Ip Address v4

IP address itu sama seperti alamat rumah yang dimana fungsinya sama sama sebagai alamat unik untuk menandakan identitas yang nanntinya digunakan untuk saling terkoneksi/tranfer data/informasi.
bentuk dari ip teridiri dari 4 octets yang dimana masing masing octetnya terdiri dari 8 bits.
masing masing octet berarti desimalnya 0.0.0.0 - 255.255.255.255.
nah angka 0 dan 255 yang diakhir ditandai sebagai network address dan broadcast address.

NAT

NAT adalah metode untuk menggunakan Public Ip address dan Private IP address.
dengan ini maka dari banyaknya Private IP address untuk mengakses internet harus melalui Public IP address.
NAT hanya menggunakan 1 Public IP Address sebagai akses internet dari banyaknya Private IP Address.
